# Bookmark App 🔖

A bookmark management application that allows users to save, organize, update, and delete website links efficiently.

🔗 Live Demo: https://malakmohamed119.github.io/Bookmark/


## ✨ Features

* Add New Bookmarks
* Edit Existing Bookmarks
* Delete Bookmarks
* LocalStorage Persistence
* Full CRUD Operations

## 🛠️ Tech Stack

* JavaScript
* Bootstrap
* HTML5
* CSS3
* LocalStorage

Built with ❤️ using JavaScript.
